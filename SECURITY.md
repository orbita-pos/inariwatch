# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in InariWatch, please report it responsibly by emailing **info@jesusbr.com**. Do not open a public issue.

Include as much detail as possible: steps to reproduce, affected component, and potential impact.

## Response Timeline

- **Acknowledgment:** within 48 hours
- **Fix:** 7-14 days depending on severity
- **Public disclosure:** 30 days after the fix is released

## Scope

The following components are in scope:

- Web application (Next.js)
- CLI tool (Rust)
- Desktop application

## Out of Scope

- Social engineering attacks against maintainers or users
- Denial of Service (DoS/DDoS) attacks
- Issues in third-party dependencies (report these upstream)
